# Portfolio Single-Page

Sitio estático listo para publicar sin build step.

## Archivos

- `index.html`: estructura principal del portfolio.
- `styles.css`: diseño visual, responsive y componentes.
- `script.js`: animaciones ligeras y helpers de contacto.

## Antes de publicar

- Reemplaza el `href="#"` del ícono de LinkedIn en `index.html` por tu URL real.
- Si quieres enlazar tu CV descargable, agrega el PDF dentro de este proyecto y crea un botón hacia ese archivo.

## Hosting gratis recomendado

- `Vercel`: subes esta carpeta y obtienes un dominio tipo `tu-proyecto.vercel.app`.
- `Netlify`: puedes hacer deploy por drag-and-drop y obtienes `tu-proyecto.netlify.app`.
- `GitHub Pages`: ideal si quieres tener el código versionado en GitHub y publicar en `usuario.github.io`.
- `Cloudflare Pages`: opción estática simple con dominio `tu-proyecto.pages.dev`.

## Dominio gratis

La opción más estable y realmente gratis es usar el subdominio incluido por el hosting. Los dominios personalizados totalmente gratis suelen ser limitados, más lentos de aprobar o menos confiables.

## Vista previa local

Puedes abrir `index.html` directamente en el navegador o levantar un servidor estático simple.
